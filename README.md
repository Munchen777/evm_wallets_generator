# EVM Wallets Generator ✨

## 🛠️ Installation

1. Clone repository:
   ```bash
   git clone https://github.com/Munchen777/evm_wallets_generator.git
   ```

2. Go to folder 📂:
   ```bash
   cd evm_wallets_generator
   ```

3. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```

## Run and follow console 

```bash
python main.py
```
